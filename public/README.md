# Bolt.DIY Enhanced - Public Assets

This directory contains static assets that are served directly.

## Files

- `manifest.json` - PWA manifest for mobile app support
- `robots.txt` - SEO and crawler instructions
- `favicon.ico` - Site favicon (add your own)

## Adding Assets

Place any static files here:
- Images
- Fonts
- Icons
- PDFs
- etc.

They will be available at the root URL:
- `/manifest.json`
- `/robots.txt`
- `/your-image.png`
